# Adventure Works Sales Dashboard (Power BI)

## Project Overview

This project presents an **interactive Power BI dashboard** built using the **Adventure Works dataset**, a popular business dataset used for analytics and business intelligence practice.

The dashboard analyzes **sales performance, revenue trends, product performance, and customer insights** to help businesses make **data-driven decisions**.

This project demonstrates my ability to perform **data modeling, dashboard design, and business analytics using Power BI**.

---

# Objectives

The main objectives of this project were:

* Analyze **sales performance across products and regions**
* Track **revenue and profit trends**
* Identify **top performing products and categories**
* Provide **interactive business insights through visual dashboards**

---

# Dashboard Features

The Power BI dashboard includes:

✔ Sales performance overview
✔ Revenue and profit analysis
✔ Product category performance
✔ Regional sales comparison
✔ Interactive filters and slicers for dynamic analysis

These features allow users to **explore business performance interactively**.
